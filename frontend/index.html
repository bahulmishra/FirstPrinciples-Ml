FirstPrinciple ML

Demystifying Machine Learning through Manual Matrix Mathematics.

FirstPrinciples ML is a full-stack engineering and ML project designed to demystify "black-box" Artificial Intelligence. Every algorithm and data-processing step in this system is hard-coded from scratch using NumPy, replacing high-level library shortcuts with manual matrix mathematics, calculus, and optimization routines.

The platform automates a rigorous machine learning pipeline, beginning with an Intelligent Ingestion Layer that handles CSV, JSON, and live API data. I implemented a custom validation engine to manage categorical encoding and enforce strict "clean data" protocols, including Z-score standardization and Correlation Pruning (r > 0.9) to eliminate redundant features before they reach the math engine.

At its core, the system executes hand-written implementations of Multi-variate Linear Regression via Gradient Descent, Soft-Margin SVM using Hinge Loss, and advanced classification through KNN, Fuzzy C-Means, and noise-resistant Possibilistic C-Means. To ensure scientific integrity, I built a custom K-Fold Cross-Validation loop and a 65:35 training split. The final output translates complex tensors into a visual Decision Surface, providing a geometric map of the model's logic and ensuring complete algorithmic explainability.
